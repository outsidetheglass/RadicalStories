What is a radical


Chinese radicals are comparable to English letters. They're written in a very specific stroke order; if they're not drawn in that stroke order, it's done wrong. So listing by the number of strokes it takes to draw a character or radical like I've done here is a common organization. In the Categories tab you'll find a listing of Chinese radicals by the number of strokes it takes to draw the radical.
